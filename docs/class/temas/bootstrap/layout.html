Layout en Bootstrap

El sistema de Layout en Bootstrap permite construir interfaces adaptables y responsivas utilizando contenedores, filas, columnas y utilidades. Bootstrap se basa en un sistema de 12 columnas que se adapta automáticamente a diferentes tamaños de pantalla mediante breakpoints.

Breakpoints, contenedores y sistema de layout

Tema Clase / Concepto Valores admitidos Descripción
Breakpoints xs <576px Tamaño extra pequeño para móviles.
sm ≥576px Pantallas pequeñas como móviles grandes.
md ≥768px Tablets o pantallas medianas.
lg ≥992px Portátiles o pantallas grandes.
xl ≥1200px Monitores grandes.
xxl ≥1400px Pantallas muy grandes.
Containers .container Responsive Contenedor centrado con ancho máximo adaptativo.
.container-fluid 100% Ocupa todo el ancho del viewport.
.container-sm ≥576px Se vuelve contenedor desde sm.
.container-md ≥768px Se vuelve contenedor desde md.
.container-lg ≥992px Se vuelve contenedor desde lg.
.container-xl ≥1200px Se vuelve contenedor desde xl.
.container-xxl ≥1400px Se vuelve contenedor desde xxl.
Grid .row Fila Crea una fila dentro del sistema grid.
.col Auto Columnas automáticas que reparten el espacio.
.row-cols-* 1–6 Define número de columnas automáticas por fila.
Nesting row dentro de una columna (col) Permite grids dentro de otros grids.
Columns .col-1 → .col-12 1–12 Define el ancho según el sistema de 12 columnas.
.col-auto auto La columna ocupa solo el ancho del contenido.
.offset-* 1–11 Desplaza columnas hacia la derecha.
.order-* 0–5, first, last Cambia el orden visual de las columnas.
.col-sm-* .col-md-* .col-lg-* .col-xl-* .col-xxl-* Responsive Columnas adaptadas según breakpoint.
.align-self-* start, center, end Alinea columnas individualmente.
Gutters .g-* 0–5 Espacio entre columnas y filas.
.gx-* 0–5 Espacio horizontal entre columnas.
.gy-* 0–5 Espacio vertical entre filas.
Utilities Display d-flex, d-grid, d-block Controla cómo se muestra un elemento.
Spacing m-*, p-* Controla márgenes y padding.
Flex utilities justify-content-*, align-items-* Control de alineación en flexbox.
Text utilities text-center, text-start, text-end Alineación de texto.
Background utilities bg-primary, bg-success Colores de fondo.
Border utilities border, border-0, rounded Control de bordes.
Z-index .z-n1 -1 Coloca elemento detrás.
.z-0 0 Z-index base.
.z-1 1 Capa baja.
.z-2 2 Capa media.
.z-3 3 Capa superior.
CSS Grid d-grid grid Activa layout CSS Grid.
gap-* 0–5 Espacio entre filas y columnas.
row-gap-* 0–5 Espacio vertical en grid.
column-gap-* 0–5 Espacio horizontal en grid.
