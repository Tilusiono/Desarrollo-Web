Tipografía en CSS

La tipografía en CSS permite controlar la apariencia del texto dentro de una página web. Mediante diferentes propiedades es posible modificar la fuente, el tamaño, el peso, la alineación, el espaciado y otros aspectos visuales del texto para mejorar la legibilidad y el diseño de la interfaz.

Propiedad Uso principal Descripción Nivel de uso
font-family Tipo de fuente Define la familia tipográfica que se utilizará en el texto. Muy usado
font-size Tamaño de texto Controla el tamaño de la fuente. Muy usado
font-weight Grosor de la fuente Define qué tan grueso o delgado se muestra el texto. Muy usado
font-style Estilo de fuente Permite mostrar texto en cursiva o normal. Usado
font-variant Variación tipográfica Permite mostrar texto en pequeñas mayúsculas. Poco usado
font Propiedad abreviada Permite definir varias propiedades de fuente en una sola línea. Usado
line-height Altura de línea Define el espacio vertical entre líneas de texto. Muy usado
letter-spacing Espacio entre letras Controla la separación entre caracteres. Usado
word-spacing Espacio entre palabras Define la distancia entre palabras. Usado
text-align Alineación del texto Alinea el texto a la izquierda, derecha, centro o justificado. Muy usado
text-transform Transformación de texto Cambia el texto a mayúsculas, minúsculas o capitalizado. Muy usado
text-decoration Decoración de texto Agrega subrayado, línea superior o tachado. Muy usado
text-indent Sangría Agrega espacio al inicio de una línea de texto. Poco usado
text-shadow Sombra de texto Agrega una sombra al texto. Usado
white-space Manejo de espacios Controla cómo se manejan los espacios y saltos de línea. Usado
direction Dirección del texto Define la dirección de escritura (izquierda a derecha o derecha a izquierda). Poco usado
unicode-bidi Control bidi Controla la dirección del texto en idiomas especiales. Poco usado
