Animaciones en CSS

Las animaciones en CSS permiten crear efectos visuales dinámicos en los elementos de una página web. Mediante diferentes propiedades es posible controlar el tiempo, la repetición, la dirección y la velocidad de los movimientos o cambios visuales. Las animaciones se definen utilizando la regla @keyframes, donde se establecen las diferentes etapas o estados de la animación.

Propiedad Uso principal Ejemplo CSS Descripción detallada
@keyframes Definir animación @keyframes mover { } Permite definir las diferentes etapas de una animación. Cada etapa representa un estado visual del elemento durante el proceso de animación.
animation-name Nombre de la animación animation-name: mover; Indica el nombre de la animación que se aplicará al elemento y que fue definida previamente con la regla @keyframes.
animation-duration Duración animation-duration: 3s; Define cuánto tiempo tarda la animación en completarse desde el inicio hasta el final.
animation-delay Retraso inicial animation-delay: 2s; Establece el tiempo de espera antes de que la animación comience después de cargarse el elemento.
animation-iteration-count Repetición animation-iteration-count: infinite; Determina cuántas veces se repetirá la animación. Puede ser un número específico o infinito.
animation-direction Dirección animation-direction: alternate; Controla si la animación se reproduce normalmente, en reversa o alternando entre ambas direcciones.
animation-timing-function Velocidad animation-timing-function: ease-in-out; Define la velocidad de cambio durante la animación, controlando cómo acelera o desacelera el movimiento.
animation-fill-mode Estado final animation-fill-mode: forwards; Define cómo se aplican los estilos antes de iniciar o después de finalizar la animación.
animation-play-state Control de reproducción animation-play-state: paused; Permite pausar o continuar una animación sin eliminarla.
animation Propiedad abreviada animation: mover 3s infinite; Permite definir varias propiedades de animación en una sola línea de código.
