Pseudo-elementos en CSS

Los pseudo-elementos en CSS permiten aplicar estilos a partes específicas de un elemento HTML, como la primera letra, la primera línea o contenido que se agrega antes o después del elemento. Gracias a los pseudo-elementos es posible modificar detalles visuales del contenido sin necesidad de cambiar la estructura del HTML.

Pseudo-elemento Ejemplo CSS Uso principal Descripción Elemento HTML
::before p::before Contenido antes Inserta contenido antes del contenido real del elemento. <p>Texto</p>
::after p::after Contenido después Inserta contenido después del contenido del elemento. <p>Texto</p>
::first-letter p::first-letter Primera letra Aplica estilos únicamente a la primera letra del texto. <p>Texto</p>
::first-line p::first-line Primera línea Aplica estilos a la primera línea visible de un párrafo. <p>Texto largo</p>
::selection p::selection Texto seleccionado Permite cambiar el color del texto cuando el usuario lo selecciona. <p>Texto seleccionable</p>
::marker li::marker Marcador de lista Permite modificar el estilo de los marcadores de listas. <li>Elemento</li>
::placeholder input::placeholder Texto de ayuda Permite cambiar el estilo del texto placeholder de los inputs. <input placeholder="Nombre">
::file-selector-button input::file-selector-button Botón de archivo Permite modificar el botón de selección de archivos. <input type="file">
::backdrop dialog::backdrop Fondo del diálogo Permite cambiar el fondo detrás de un elemento <dialog>. <dialog>
::cue video::cue Subtítulos Permite aplicar estilos a los subtítulos de un video. <video>
::cue-region video::cue-region Región de subtítulos Define el estilo del área donde aparecen los subtítulos. <video>
::part() my-element::part(header) Web Components Permite estilizar partes internas de componentes personalizados. <my-element>
::slotted() slot::slotted(p) Slots en Web Components Permite aplicar estilos a elementos insertados en slots. <slot>
::highlight() p::highlight() Texto resaltado Permite aplicar estilos a texto resaltado mediante API de highlight. <p>Texto</p>
::spelling-error p::spelling-error Error ortográfico Permite mostrar estilos en palabras con errores ortográficos. <p>Texto</p>
::grammar-error p::grammar-error Error gramatical Permite mostrar estilos en errores gramaticales. <p>Texto</p>
