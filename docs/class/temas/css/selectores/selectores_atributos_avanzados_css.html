Selectores de atributo en CSS

Los selectores de atributo en CSS permiten aplicar estilos a los elementos HTML según los atributos que poseen o según el valor de dichos atributos. Esto permite seleccionar elementos de manera más específica dentro de una página web sin necesidad de utilizar clases o identificadores adicionales.

Selector Ejemplo Uso principal Descripción Elemento HTML
[atributo] input[type] Atributo existente Selecciona todos los elementos que contienen ese atributo. <input type="text">
[atributo="valor"] input[type="text"] Valor exacto Selecciona elementos cuyo atributo tiene exactamente ese valor. <input type="text">
[atributo~="valor"] div[class~="activo"] Palabra dentro de lista Selecciona elementos cuyo atributo contiene una palabra específica separada por espacios. <div class="activo principal">
[atributo^="valor"] a[href^="https"] Empieza con Selecciona elementos cuyo atributo comienza con el valor indicado. <a href="https://sitio.com">
[atributo$="valor"] img[src$=".png"] Termina con Selecciona elementos cuyo atributo termina con el valor indicado. <img src="imagen.png">
[atributo*="valor"] div[class*="card"] Contiene texto Selecciona elementos cuyo atributo contiene una parte del valor. <div class="card-producto">
[atributo|="valor"] p[lang|="es"] Valor o valor-guion Selecciona elementos cuyo atributo es ese valor o comienza con él seguido de guion. <p lang="es-PE">
