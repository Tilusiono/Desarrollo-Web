Combinadores en CSS

Los combinadores en CSS permiten seleccionar elementos según la relación que tienen entre sí dentro del documento HTML. Gracias a estos selectores podemos aplicar estilos dependiendo de la estructura de la página, como elementos hijos, descendientes o hermanos.

Combinador Símbolo Uso principal Ejemplo CSS Descripción
Descendente (espacio) Selecciona elementos dentro de otro div p Aplica estilos a todos los párrafos dentro del div.
Hijo directo > Selecciona hijos inmediatos div > p Aplica estilos solo a los párrafos que son hijos directos.
Hermano adyacente + Selecciona el primer hermano siguiente h2 + p Selecciona el primer párrafo que aparece después del h2.
Hermanos generales ~ Selecciona todos los hermanos posteriores h2 ~ p Selecciona todos los párrafos que aparecen después del h2.
