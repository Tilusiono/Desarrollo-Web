Selectores lógicos modernos en CSS

Los selectores lógicos modernos en CSS permiten seleccionar elementos de manera más flexible mediante condiciones, agrupaciones o relaciones entre elementos dentro del documento HTML. Estos selectores forman parte de los Selectors Level 4 y facilitan escribir código más limpio, reutilizable y con menor repetición de reglas CSS.

Selector lógico Ejemplo CSS Uso principal Descripción Elemento HTML
:not() p:not(.activo) Excluir elementos Selecciona todos los elementos que no cumplen una condición específica. <p class="texto">
:is() :is(h1, h2, h3) Agrupar selectores Permite agrupar varios selectores en una sola regla CSS para evitar repetir código. <h1>Título</h1>
:where() :where(section, article) Agrupación sin especificidad Funciona similar a :is(), pero no aumenta la especificidad del selector. <section>Contenido</section>
:has() div:has(img) Selector relacional Selecciona un elemento si contiene otro elemento dentro de él. <div><img src="foto.jpg"></div>
:matches() :matches(h1, h2) Agrupación antigua Versión antigua de :is(), utilizada en implementaciones previas de CSS. <h1>Título</h1>
