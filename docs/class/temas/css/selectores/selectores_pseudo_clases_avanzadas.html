Pseudo-clases en CSS

Las pseudo-clases en CSS permiten aplicar estilos a los elementos HTML según su estado, posición o interacción dentro de la página. Estas pseudo-clases permiten modificar la apariencia de los elementos cuando el usuario interactúa con ellos o cuando cumplen ciertas condiciones, como ser el primer elemento, estar enfocado o encontrarse seleccionado en un formulario.

Pseudo-clase Ejemplo CSS Uso principal Descripción Elemento HTML
:hover a:hover Interacción con mouse Aplica estilos cuando el cursor pasa sobre el elemento. <a>Enlace</a>
:active button:active Elemento presionado Se activa cuando el usuario hace clic sobre el elemento. <button>Enviar</button>
:focus input:focus Elemento enfocado Se aplica cuando un elemento recibe foco. <input type="text">
:first-child p:first-child Primer hijo Selecciona el primer elemento hijo dentro de un contenedor. <p>Texto</p>
:last-child p:last-child Último hijo Selecciona el último elemento hijo dentro de su contenedor. <p>Texto</p>
:nth-child() li:nth-child(2) Posición específica Selecciona elementos según su posición. <li>Item</li>
:nth-last-child() li:nth-last-child(2) Posición desde el final Selecciona elementos contando desde el final. <li>Item</li>
:first-of-type p:first-of-type Primer tipo Selecciona el primer elemento de ese tipo. <p>Texto</p>
:last-of-type p:last-of-type Último tipo Selecciona el último elemento de ese tipo. <p>Texto</p>
:nth-of-type() p:nth-of-type(3) Tipo específico Selecciona elementos según tipo y posición. <p>Texto</p>
:not() p:not(.activo) Excluir elementos Selecciona elementos que no cumplen la condición. <p class="texto">
:is() :is(h1,h2,h3) Agrupar selectores Permite aplicar estilos a varios selectores. <h1>Título</h1>
:where() :where(div,p) Selector flexible Similar a :is pero con menor especificidad. <div>Contenido</div>
:has() div:has(img) Elemento con contenido Selecciona elementos que contienen otro elemento. <div><img></div>
:checked input:checked Formulario marcado Selecciona checkboxes o radios seleccionados. <input type="checkbox">
:disabled input:disabled Campo deshabilitado Selecciona campos que están deshabilitados. <input disabled>
:enabled input:enabled Campo habilitado Selecciona elementos de formulario activos. <input>
:required input:required Campo obligatorio Selecciona campos que requieren información. <input required>
:valid input:valid Formulario válido Selecciona campos con contenido válido. <input type="email">
:invalid input:invalid Formulario inválido Selecciona campos con contenido incorrecto. <input type="email">
