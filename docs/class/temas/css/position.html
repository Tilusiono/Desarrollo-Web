Position en CSS

La propiedad position en CSS define cómo se posiciona un elemento dentro de la página. Permite controlar la ubicación de los elementos utilizando propiedades como top, right, bottom y left. Dependiendo del valor utilizado, el elemento puede mantenerse en el flujo normal del documento o posicionarse de forma independiente.

Propiedad / Valor Tipo Uso principal Nivel de uso Descripción
position Propiedad principal Tipo de posicionamiento Muy usado Define el método de posicionamiento de un elemento.
static Valor de position Posición normal Muy usado Valor por defecto. El elemento sigue el flujo normal del documento.
relative Valor de position Posición relativa Muy usado Permite mover el elemento respecto a su posición original.
absolute Valor de position Posición absoluta Muy usado El elemento se posiciona respecto a su contenedor posicionado más cercano.
fixed Valor de position Posición fija Muy usado El elemento se fija respecto al viewport del navegador.
sticky Valor de position Posición pegajosa Usado Actúa como relative hasta que alcanza un límite y luego se fija.
top Propiedad relacionada Distancia superior Muy usado Define la distancia desde el borde superior del contenedor.
right Propiedad relacionada Distancia derecha Muy usado Define la distancia desde el borde derecho.
bottom Propiedad relacionada Distancia inferior Muy usado Define la distancia desde el borde inferior.
left Propiedad relacionada Distancia izquierda Muy usado Define la distancia desde el borde izquierdo.
inset Propiedad moderna Atajo de top/right/bottom/left Usado Permite definir las cuatro posiciones en una sola propiedad.
z-index Propiedad relacionada Orden de capas Muy usado Controla qué elemento aparece encima de otro.
inset-block Propiedad lógica Top + bottom lógico Poco usado Define posiciones verticales en layouts lógicos.
inset-inline Propiedad lógica Left + right lógico Poco usado Define posiciones horizontales en layouts lógicos.
