Unidades en CSS

Las unidades en CSS permiten definir tamaños, proporciones y espacios dentro de un diseño web. Algunas unidades son absolutas, mientras que otras son relativas al tamaño del texto, al contenedor o al tamaño de la pantalla. Elegir la unidad correcta es importante para lograr diseños flexibles y adaptables a diferentes dispositivos.

Unidad Tipo Uso principal Ejemplo CSS Nivel de uso Descripción
px Absoluta Tamaños fijos width: 200px; Muy usada Representa píxeles en pantalla y es una de las unidades más utilizadas.
% Relativa Proporciones width: 50%; Muy usada Se calcula en relación al tamaño del elemento contenedor.
em Relativa Tamaño de texto font-size: 1.5em; Muy usada Relativa al tamaño de fuente del elemento padre.
rem Relativa Tamaño global font-size: 1.2rem; Muy usada Relativa al tamaño de fuente del elemento raíz (html).
vw Viewport Ancho de pantalla width: 50vw; Muy usada Equivale al 1% del ancho de la ventana del navegador.
vh Viewport Altura de pantalla height: 100vh; Muy usada Equivale al 1% de la altura del viewport.
vmin Viewport Escalado flexible font-size: 5vmin; Usada Usa el valor menor entre el ancho y alto de la pantalla.
vmax Viewport Escalado flexible font-size: 5vmax; Usada Usa el valor mayor entre el ancho y alto del viewport.
ch Relativa Ancho de texto width: 40ch; Usada Representa el ancho del carácter "0".
ex Relativa Altura tipográfica font-size: 2ex; Poco usada Basada en la altura de la letra "x".
cm Absoluta Medidas físicas width: 10cm; Poco usada Unidad basada en centímetros.
mm Absoluta Medidas físicas width: 20mm; Poco usada Unidad basada en milímetros.
in Absoluta Medidas físicas width: 2in; Poco usada Unidad basada en pulgadas.
pt Absoluta Tipografía font-size: 12pt; Poco usada Unidad tradicional usada en impresión.
pc Absoluta Tipografía font-size: 1pc; Poco usada Equivale a 12 puntos tipográficos.
svh Viewport moderno Altura mínima viewport height: 100svh; Nueva Representa la altura mínima del viewport en dispositivos móviles.
lvh Viewport moderno Altura máxima viewport height: 100lvh; Nueva Representa la altura máxima del viewport.
dvh Viewport moderno Viewport dinámico height: 100dvh; Nueva Se ajusta dinámicamente al viewport visible.
svw Viewport moderno Ancho mínimo viewport width: 100svw; Nueva Representa el ancho mínimo del viewport.
lvw Viewport moderno Ancho máximo viewport width: 100lvw; Nueva Representa el ancho máximo del viewport.
dvw Viewport moderno Ancho dinámico viewport width: 100dvw; Nueva Se ajusta dinámicamente al ancho visible del viewport.
