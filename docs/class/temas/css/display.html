Tipo de visualización - Display en CSS

La propiedad display en CSS define cómo se comporta un elemento dentro del flujo del documento. Determina si un elemento se muestra como bloque, en línea, como contenedor flexible o como una cuadrícula. Es una de las propiedades más importantes para controlar la estructura y el layout de una página web.

Valor Uso principal Nivel de uso Descripción
block Elemento en bloque Muy usado Ocupa todo el ancho disponible y comienza en una nueva línea.
inline Elemento en línea Muy usado Solo ocupa el espacio necesario y no inicia una nueva línea.
inline-block Bloque en línea Muy usado Se comporta como inline pero permite width y height.
none Ocultar elemento Muy usado El elemento desaparece completamente del layout.
flex Contenedor flexible Muy usado Permite crear layouts flexibles con alineación automática.
inline-flex Flex en línea Usado Similar a flex pero el contenedor se comporta como inline.
grid Layout en cuadrícula Muy usado Permite crear estructuras de filas y columnas.
inline-grid Grid en línea Usado Grid pero con comportamiento inline.
flow-root Nuevo contexto de bloque Moderado Crea un nuevo contexto de bloque y evita problemas con floats.
contents Eliminar contenedor visual Poco usado El contenedor desaparece pero sus hijos permanecen en el DOM.
list-item Elemento de lista Usado Hace que un elemento se comporte como un item de lista con marcador.
table Simular tabla Poco usado Hace que el elemento se comporte como una tabla HTML.
table-row Fila de tabla Poco usado Representa una fila dentro de una tabla.
table-cell Celda de tabla Poco usado Representa una celda dentro de una tabla.
table-row-group Grupo de filas Muy poco usado Agrupa varias filas dentro de una tabla.
table-header-group Encabezado de tabla Muy poco usado Define el grupo de encabezado de una tabla.
table-footer-group Pie de tabla Muy poco usado Define el pie de una tabla.
table-column Columna de tabla Muy poco usado Define propiedades para una columna.
table-column-group Grupo de columnas Muy poco usado Agrupa varias columnas.
table-caption Título de tabla Muy poco usado Define el título o caption de una tabla.
ruby Tipografía asiática Casi no usado Usado en anotaciones tipográficas para idiomas asiáticos.
ruby-base Base ruby Casi no usado Parte base del sistema ruby.
ruby-text Texto ruby Casi no usado Texto auxiliar en anotaciones ruby.
ruby-base-container Contenedor ruby Casi no usado Contenedor para texto base ruby.
ruby-text-container Contenedor ruby Casi no usado Contenedor para texto auxiliar ruby.
