CSS Grid completo

CSS Grid es un sistema de layout bidimensional que permite organizar elementos en filas y columnas dentro de un contenedor. Ofrece control preciso sobre la distribución, alineación y tamaño de los elementos dentro de la cuadrícula.

Propiedad / Valor Tipo Uso principal Nivel de uso Descripción
display: grid Contenedor Activar Grid Muy usado Convierte el elemento en un contenedor grid.
display: inline-grid Contenedor Grid en línea Usado Hace que el contenedor grid se comporte como inline.
grid-template-columns Contenedor Definir columnas Muy usado Define el número y tamaño de columnas.
grid-template-rows Contenedor Definir filas Muy usado Define el número y tamaño de filas.
grid-template-areas Contenedor Layout por áreas Usado Permite nombrar áreas dentro de la cuadrícula.
grid-template Contenedor Shorthand Usado Combina grid-template-rows, columns y areas.
grid-auto-columns Contenedor Columnas automáticas Poco usado Tamaño de columnas generadas automáticamente.
grid-auto-rows Contenedor Filas automáticas Poco usado Tamaño de filas generadas automáticamente.
grid-auto-flow Contenedor Flujo automático Usado Controla cómo se colocan automáticamente los elementos.
row-gap Contenedor Espacio entre filas Muy usado Define la separación vertical entre filas.
column-gap Contenedor Espacio entre columnas Muy usado Define la separación horizontal entre columnas.
gap Contenedor Separación general Muy usado Define espacio entre filas y columnas.
justify-items Contenedor Alineación horizontal Usado Alinea los elementos dentro de su celda horizontalmente.
align-items Contenedor Alineación vertical Usado Alinea los elementos verticalmente dentro de su celda.
place-items Contenedor Shorthand Usado Combina justify-items y align-items.
justify-content Contenedor Alinear cuadrícula Usado Alinea toda la cuadrícula horizontalmente.
align-content Contenedor Alinear verticalmente Usado Alinea toda la cuadrícula verticalmente.
place-content Contenedor Shorthand Usado Combina justify-content y align-content.
grid-column-start Ítem Inicio columna Usado Define en qué columna inicia el elemento.
grid-column-end Ítem Fin columna Usado Define dónde termina la columna.
grid-column Ítem Shorthand columnas Muy usado Combina column-start y column-end.
grid-row-start Ítem Inicio fila Usado Define en qué fila comienza.
grid-row-end Ítem Fin fila Usado Define dónde termina.
grid-row Ítem Shorthand filas Muy usado Combina row-start y row-end.
grid-area Ítem Área completa Muy usado Define ubicación completa o conecta con grid-template-areas.
justify-self Ítem Alineación horizontal individual Usado Alinea un elemento dentro de su celda.
align-self Ítem Alineación vertical individual Usado Alinea un elemento verticalmente.
place-self Ítem Shorthand Poco usado Combina align-self y justify-self.
subgrid Valor Subcuadrícula Poco usado Permite heredar la estructura grid del contenedor padre.
repeat() Función Repetir columnas o filas Muy usado Permite definir múltiples columnas o filas de forma compacta.
minmax() Función Tamaño flexible Muy usado Define un tamaño mínimo y máximo para filas o columnas.
auto-fit Valor Columnas responsivas Usado Adapta columnas automáticamente según espacio disponible.
auto-fill Valor Columnas automáticas Usado Rellena la cuadrícula con columnas automáticas.
