Modelo de Caja en CSS

El modelo de caja (Box Model) es un concepto fundamental en CSS que describe cómo se estructura y calcula el espacio que ocupa un elemento en una página web. Cada elemento HTML se representa como una caja compuesta por cuatro partes: contenido (content), padding, borde (border) y margen (margin). Estas áreas determinan el tamaño total del elemento y la separación que tendrá respecto a otros elementos dentro del diseño.
