Variables en CSS

Las variables en CSS, también conocidas como Custom Properties, permiten almacenar valores reutilizables dentro de una hoja de estilos. Gracias a ellas es posible reutilizar colores, tamaños, fuentes u otros valores en diferentes partes del diseño, facilitando el mantenimiento y la consistencia del código. Las variables se definen utilizando dos guiones (--) y se utilizan mediante la función var().

Concepto Uso principal Ejemplo CSS Descripción detallada
:root Variables globales :root { --color: blue; } Permite definir variables globales que pueden ser utilizadas en todo el documento.
--variable Declarar variable --color-principal: red; Las variables en CSS se declaran usando dos guiones al inicio del nombre.
var() Usar variable color: var(--color-principal); La función var() permite acceder al valor almacenado en una variable.
var(valor, fallback) Valor alternativo color: var(--color, black); Permite definir un valor alternativo si la variable no existe.
Variables locales Variables dentro de un selector .caja { --tamano: 20px; } Las variables también pueden definirse dentro de un selector y solo se aplican en ese contexto.
Reutilización Uso repetido background: var(--color-principal); Permite reutilizar un mismo valor en múltiples propiedades.
Variables en tamaños Control de tamaños font-size: var(--tamano-texto); Las variables pueden almacenar tamaños para textos, márgenes o padding.
Variables en colores Paleta de colores background: var(--color-secundario); Muy utilizadas para mantener consistencia en la paleta de colores de un diseño.
Variables en cálculos Uso con calc() width: calc(var(--ancho) * 2); Las variables pueden combinarse con funciones como calc() para realizar cálculos.
Herencia Propagación de variables color: var(--color-principal); Las variables CSS se heredan a los elementos hijos dentro del DOM.
