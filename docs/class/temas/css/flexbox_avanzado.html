Flexbox avanzado en CSS

Flexbox (Flexible Box Layout) es un sistema de diseño en CSS que permite organizar y alinear elementos dentro de un contenedor flexible. Proporciona herramientas para distribuir espacio, controlar alineaciones en dos ejes y adaptar el layout a diferentes tamaños de pantalla.

Propiedad / Valor Tipo Uso principal Nivel de uso Descripción
display: flex Contenedor Activar Flexbox Muy usado Convierte el elemento en un contenedor flex y permite controlar sus elementos hijos.
display: inline-flex Contenedor Flex en línea Usado Hace que el contenedor use Flexbox pero se comporte como elemento inline.
flex-direction Contenedor Dirección de elementos Muy usado Define la dirección del eje principal donde se colocan los elementos.
row Valor Dirección horizontal Muy usado Los elementos se organizan en fila de izquierda a derecha.
row-reverse Valor Fila inversa Usado Los elementos se colocan en fila pero en orden inverso.
column Valor Dirección vertical Muy usado Los elementos se organizan en columna.
column-reverse Valor Columna inversa Usado Los elementos se organizan verticalmente en orden inverso.
flex-wrap Contenedor Salto de línea Muy usado Permite que los elementos pasen a una nueva línea cuando no hay espacio.
nowrap Valor Una sola línea Muy usado Los elementos permanecen en una sola fila.
wrap Valor Varias líneas Muy usado Los elementos pueden pasar a otra línea.
wrap-reverse Valor Líneas inversas Poco usado Las nuevas filas se colocan en dirección inversa.
flex-flow Contenedor Shorthand Usado Combina flex-direction y flex-wrap.
justify-content Contenedor Alineación eje principal Muy usado Distribuye los elementos a lo largo del eje principal.
flex-start Valor Inicio Muy usado Los elementos se alinean al inicio del contenedor.
flex-end Valor Final Muy usado Los elementos se alinean al final del contenedor.
center Valor Centrado Muy usado Los elementos se centran dentro del contenedor.
space-between Valor Espacio entre Muy usado Distribuye espacio entre los elementos.
space-around Valor Espacio alrededor Usado Agrega espacio alrededor de cada elemento.
space-evenly Valor Espacio uniforme Usado El espacio entre elementos es uniforme.
align-items Contenedor Alineación eje cruzado Muy usado Alinea los elementos verticalmente u horizontalmente según el eje.
stretch Valor Estirar elementos Muy usado Los elementos ocupan todo el espacio disponible.
baseline Valor Alineación de texto Poco usado Alinea los elementos según la línea base del texto.
align-content Contenedor Alinear múltiples líneas Usado Controla la distribución cuando hay varias filas de elementos.
gap Contenedor Espacio entre elementos Muy usado Define separación entre elementos.
row-gap Contenedor Espacio entre filas Usado Define el espacio vertical entre filas.
column-gap Contenedor Espacio entre columnas Usado Define el espacio horizontal entre columnas.
order Ítem Cambiar orden Usado Permite cambiar el orden visual de los elementos.
flex-grow Ítem Expandir elemento Muy usado Define cuánto puede crecer un elemento.
flex-shrink Ítem Reducir elemento Usado Define cuánto puede reducirse si falta espacio.
flex-basis Ítem Tamaño base Muy usado Define el tamaño inicial antes de distribuir el espacio.
flex Ítem Shorthand Muy usado Combina flex-grow, flex-shrink y flex-basis.
align-self Ítem Alineación individual Usado Permite cambiar la alineación de un solo elemento.
