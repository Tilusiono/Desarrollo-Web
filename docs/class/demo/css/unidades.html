Demo de las 21 unidades CSS

En esta página se muestran ejemplos visuales de unidades absolutas, relativas, viewport y viewport modernas en CSS.

1. px

Ancho en píxeles

Unidad absoluta basada en píxeles de pantalla. No depende de otros elementos. Se recomienda usarla cuando se necesita un tamaño fijo como bordes, íconos o imágenes.

2. %

Ancho en porcentaje

Ancho en porcentaje

Se calcula respecto al tamaño del elemento contenedor. Por ejemplo, 50% significa la mitad del ancho del elemento padre. Se usa mucho para layouts flexibles.

3. em

Texto con tamaño en em

Se calcula en relación al tamaño de fuente del elemento padre. Por ejemplo, si el padre tiene 16px, entonces 2em equivale a 32px. Se usa para tipografía y componentes escalables.

4. rem

Texto con tamaño en rem

Se calcula respecto al tamaño de fuente del elemento raíz (html). Normalmente 1rem equivale a 16px. Es muy recomendable para mantener escalabilidad consistente en todo el sitio.

5. vw

Ancho en vw

Representa el 1% del ancho de la ventana del navegador. Por ejemplo, 50vw significa la mitad del ancho de la pantalla. Muy usado en diseño responsive.

6. vh

Altura en vh

Representa el 1% de la altura del viewport (pantalla). Por ejemplo, 50vh ocupa la mitad la altura visible del navegador. Muy usado para secciones full screen.

7. vmin

Texto en vmin

Se calcula tomando el valor menor entre el ancho y la altura del viewport. Es útil para crear elementos que se adapten proporcionalmente a la pantalla.

8. vmax

Texto en vmax

Se calcula tomando el valor mayor entre el ancho y la altura del viewport. Se usa cuando se quiere que un elemento escale según la dimensión más grande de la pantalla.

9. ch


Se basa en el ancho del carácter "0" de la fuente actual. Muy útil para limitar la longitud de texto o formularios según número de caracteres.

10. ex

Texto con tamaño en ex

Se calcula según la altura de la letra "x" de la tipografía utilizada. No se usa mucho porque depende mucho de la fuente.

11. cm

Unidad física basada en centímetros. Se usa principalmente en impresión, no en diseño web responsivo.

12. mm

Unidad física basada en milímetros. Similar a cm y utilizada sobre todo en documentos destinados a impresión.

13. in

Unidad basada en pulgadas (1in = 2.54cm). Usada principalmente en diseño de impresión o documentos.

14. pt

Texto con tamaño en puntos

Unidad tipográfica usada tradicionalmente en impresión. 1 punto equivale a 1/72 de pulgada.

15. pc

Texto con tamaño en picas

Unidad tipográfica donde 1pc equivale a 12pt. Se utiliza en tipografía profesional e impresión.

16. svh

Caja con altura en svh

Representa el 1% de la altura mínima del viewport en dispositivos móviles. Evita problemas cuando aparece la barra del navegador.

  • viewport = área visible del navegador
  • s = small (altura mínima)
  • vh = 1% de la altura del viewport

17. lvh

Caja con altura en lvh

Representa el 1% de la altura máxima del viewport. Se usa para calcular la altura total posible del navegador.

  • viewport = área visible del navegador
  • l = large ( altura máxima )
  • vh = 1% de la altura del viewport

18. dvh

Caja con altura en dvh

Viewport dinámico que se adapta cuando cambia el tamaño visible del navegador. Es una unidad moderna recomendada para layouts móviles.

  • viewport = área visible del navegador
  • dynamic = cambia si aparece o desaparece la barra del navegador
  • vh = 1% de la altura del viewport

19. svw

Representa el ancho mínimo del viewport en dispositivos móviles. Permite controlar cambios provocados por barras del navegador.

  • viewport = área visible del navegador
  • s = small (altura mínima)
  • vw = 1% del ancho del viewport

20. lvw

Representa el ancho máximo del viewport disponible. Se usa para cálculos de layout que necesitan el ancho completo.

  • viewport = área visible del navegador
  • l = large ( altura máxima )
  • vw = 1% del ancho del viewport

21. dvw

Unidad dinámica que representa el ancho visible del viewport y se adapta cuando el tamaño de la ventana cambia.

  • viewport = área visible del navegador
  • dynamic = cambia si aparece o desaparece la barra del navegador
  • vw = 1% del ancho del viewport
